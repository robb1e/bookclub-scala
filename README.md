bookclub-scala
==============

Some examples of Scala while reading "Seven Languages in Seven weeks" for Bookclub.

## Setup

  brew install scala

## Running

  scala **.scala

## Examples

* Case classes
* Currying
* Traits and Duck Typing
* High Order Functions
* List comprehension
* Pattern Matching
* Implicit Parameters
* Implicit Conversions
* Operators
* Types
